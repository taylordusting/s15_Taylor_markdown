## What is Markdown?
A type of markup language
* Plain text formatting
* Easily converted to HTML
* Plain text --> Rich text

Two Types:
* Standard Markdown (SM)
* GitHub Flavored Markdown (GFM)

## What can you do with it?
* Styling
* Word formatting
* Add images
* Create Lists (ordered/unordered)
* Links
* Code blocks
